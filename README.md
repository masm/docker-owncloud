# docker-owncloud

> A ownCloud 7 image.

## Install

Pull this image with `docker pull masm/owncloud`.

## Usage

Nginx runs on port 80. Make sure you expose that port.

Data is stored at /var/www/owncloud/data and the configuration is stored at /var/www/owncloud/config.
